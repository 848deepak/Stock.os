# User Personas

## Admin

### Goal

Run the system reliably and maintain inventory governance.

### Needs

1. Full access across entities and controls
2. Reliable metrics and audit visibility
3. Simple recovery path during deployment issues

## Operations Manager

### Goal

Keep inventory healthy and avoid stock disruption.

### Needs

1. Fast visibility into low stock and trend signals
2. Easy transaction operations
3. Clear category and warehouse drilldown

## Staff

### Goal

Execute operational tasks quickly and correctly.

### Needs

1. Minimal UI friction
2. Safe role-scoped access
3. Barcode-assisted lookup and updates

## Teacher Demo Persona

### Goal

Review product capabilities quickly in a constrained demo session.

### Needs

1. Stable demo credentials
2. Realistic seeded data to verify charts and workflows
3. Predictable feature behavior without setup overhead
